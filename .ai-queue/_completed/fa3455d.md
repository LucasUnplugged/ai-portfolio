**Implementation:**

1. **Project header:**
   - Back link "вҶҗ Dashboard" pointing to `/app/ledger` (use lucide-react `ArrowLeft`)
   - Project name (large, `font-heading`)
   - Status badge using `Badge` component
   - Description text
   - Member avatars row (use `Avatar` component with initials)

2. **Tabbed content:** Use shadcn `Tabs` component with 3 tabs:

   **Documents tab (default):**
   - List of documents belonging to this project (filter `documents` by projectId)
   - Each row: document title, status badge (draft/published/archived), last edited by (user name), last edited time
   - Use a simple list layout (not a full table) вҖ” each document is a clickable-looking row with subtle hover state
   - Link each document to `/app/ledger/document` (since we only have one document page)

   **Tasks tab:**
   - Summary section: 4 mini stat cards showing count per status (Backlog, In Progress, In Review, Done)
   - Below: list of tasks for this project, sorted by priority (urgent first)
   - Each task: use `TaskCard` component
   - Grid layout: 2 columns on desktop, 1 on mobile

   **Activity tab:**
   - Use `ActivityFeed` component filtered to this project's entries
   - Since audit entries reference targetType/targetId, filter to entries where targetId matches a document or task in this project

3. **Data:** Use project ID "proj-1" (Brand Redesign). Import from `@/data/ledger`.

- Use `Badge` for document and project statuses
